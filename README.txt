Portfolio split files

This version uses only HTML, CSS, and JavaScript.
No Tailwind CSS, no Font Awesome, and no external JavaScript framework are required.

Files:
1. index.html  - Main website structure and text content
2. styles.css  - All design, layout, color, responsive CSS
3. script.js   - Modal, copy button, mobile menu, skill bar logic

How to edit:
- Edit text/content: open index.html
- Edit color/layout/style: open styles.css
- Edit demo links/login credentials: open script.js, then edit the projects object at the top

How to run:
- Open index.html in a browser.
